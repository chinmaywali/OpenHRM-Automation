# OpenHRM-Automation

This project is based on Python-Selenium to automate https://opensource-demo.orangehrmlive.com/  website. 
Functionality covered in this automation Project :

- Login Functionality
- About section 
- Support section
- Help section
- Forgot Password Functionality
- Logout Functionality
